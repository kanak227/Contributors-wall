# Contributing to the Contributors Wall

We are happy to have you here! To add your profile to the wall, follow these steps:

1. **Fork** this repository.
2. Create a new file in the `contributors/` directory named `your-username.json`.
3. Use the following template:
   ```json
   {
     "name": "Your Name",
     "username": "your-github-username",
     "domain": "e.g. Full Stack Developer",
     "color": "blue"
   }
   ```
   *Allowed colors: red, blue, green, purple, orange*
4. Commit your changes and create a **Pull Request**.

### What happens next?
Once your PR is merged, a GitHub Action will automatically update the contributors list, and your profile will appear on the wall!
